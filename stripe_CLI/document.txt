1. Open PowerShell in the Stripe CLI Directory
	Navigate to the folder where stripe.exe is located. Right-click inside the folder and choose:
	Open in Terminal or Open PowerShell window here.

2. Log in to Stripe
	Run the following command:
	.\stripe.exe login
	
	This will prompt you to open a browser page. Press Enter when asked, and log in to your Stripe account.

3. Authorize the CLI
	After logging in, a pairing code will appear in the PowerShell window. Verify the same code in your browser and click Allow to complete the pairing.
	To begin listening to Stripe webhook events, run:
	.\stripe.exe listen
